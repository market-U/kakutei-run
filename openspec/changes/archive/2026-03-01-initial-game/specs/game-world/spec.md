# Spec: game-world

## ADDED Requirements

### Requirement: 視差スクロール

ゲーム世界はプレーヤーを中央に固定し、3レイヤーの背景が異なる速度でスクロールする。

#### Scenario: 各レイヤーのスクロール速度

- **WHEN** ゲームが進行中である
- **THEN** 遠景（bg_far）はスクロール速度×0.3、近景（bg_near）は×0.7、地面は×1.0 で左にスクロールする

#### Scenario: 背景のループ

- **WHEN** 背景画像が画面左端から外れる
- **THEN** 同じ画像が右端に再配置され、シームレスなループが維持される

---

### Requirement: ランダムマップ生成

ゲーム開始ごとにプレイ内容が異なるよう、難易度設定に従ってマップが自動生成される。

#### Scenario: 難易度に応じたレシート配置

- **WHEN** ゲームが開始される
- **THEN** 難易度に対応する枚数（Easy:20枚/Normal:35枚/Hard:55枚）のレシートがステージ全体にランダム配置される

#### Scenario: 難易度に応じた石ころ配置

- **WHEN** ゲームが開始される
- **THEN** 難易度に応じた密度（低/中/高）で石ころがランダム配置される

#### Scenario: 難易度に応じた魔女配置

- **WHEN** ゲームが開始される
- **THEN** 難易度に応じた頻度（低/中/高）で魔女がランダム配置される

#### Scenario: ゴール地点の固定

- **WHEN** マップが生成される
- **THEN** 税務署（ゴール）はステージ末端に固定配置される

#### Scenario: プレーヤー周辺の安全地帯

- **WHEN** マップが生成される
- **THEN** ゲーム開始時の初期画面全体（worldX < CANVAS_W の領域）には障害物が配置されない
- **AND** すべての障害物・アイテムの worldX は CANVAS_W（960px）以上となり、スクロール開始後に画面右から出現する
