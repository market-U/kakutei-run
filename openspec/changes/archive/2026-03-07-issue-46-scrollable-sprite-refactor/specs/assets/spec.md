# Spec Delta: assets

## MODIFIED Requirements

### Requirement: スプライトシートの仕様

ゲームで使用するアニメーションはスプライトシート形式（横並びの連番フレーム）で管理される。
アニメーションがない場合でも、将来の追加に備えて1フレームのスプライトシートとして管理できる。

#### Scenario: スプライトシートの読み込み

- **WHEN** BootScene でアセットが読み込まれる
- **THEN** 以下のスプライトシートが所定のアセットキーで登録される
  - 自キャラ走行（6〜8フレーム）
  - 自キャラ転倒（4〜6フレーム）
  - 自キャラゴール（4〜6フレーム）
  - 自キャラ腰痛走行（6〜8フレーム）
  - 敵キャラ走行（6〜8フレーム）
  - 魔女ふわふわ（2〜4フレーム）
  - レシート（1フレーム、32 × 48 px）

### Requirement: 静止画アセットの仕様

アニメーション不要かつスプライトシートとして管理しないオブジェクトは単体画像で管理される。

#### Scenario: 静止画アセットの読み込み

- **WHEN** BootScene でアセットが読み込まれる
- **THEN** 以下の静止画が所定のアセットキーで登録される
  - 税務署（ゴール）
  - 石ころ（1〜3種類）
  - 背景遠景（bg_far、幅2048px以上）
  - 背景近景（bg_near、幅2048px以上）
  - 地面タイル（横ループパターン）
