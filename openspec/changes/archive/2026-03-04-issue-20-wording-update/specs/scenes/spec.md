# Delta Spec: scenes

## MODIFIED Requirements

### Requirement: タイトルシーン

ゲーム起動時に表示されるシーンで難易度選択とゲーム開始ができる。縦長キャンバスのゾーン構成に合わせたレイアウトで表示する。

#### Scenario: タイトル表示

- **WHEN** ゲームが起動する
- **THEN** UI上部ゾーン（y=0〜360）にタイトル名とサブテキスト（難易度選択の案内）が表示される
- **AND** UI下部ゾーン（y=900〜1440）に各難易度の `displayName` を表示した選択ボタンが**縦並び**で表示される
- **AND** UI下部ゾーンにスタートボタンが表示される

#### Scenario: 難易度選択

- **WHEN** プレーヤーが難易度を選択する
- **THEN** 選択状態がハイライトされ、その難易度でゲームが開始できる状態になる

#### Scenario: ゲーム開始

- **WHEN** プレーヤーがスタートボタンを押す
- **THEN** 選択済み難易度でゲームシーンに遷移する
