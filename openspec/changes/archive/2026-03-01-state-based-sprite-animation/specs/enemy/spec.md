# Spec: enemy

## ADDED Requirements

### Requirement: プレーヤー接触後のアニメーション停止

敵キャラクターがプレーヤーに到達（接触）した後、走行アニメーションを停止し、スプライトを現在のフレームに固定する。

#### Scenario: 敵がプレーヤーに到達した後のアニメーション

- **WHEN** 敵キャラクターの `stopChasing()` が呼ばれる
- **THEN** 走行アニメーション（`enemy_run`）が停止し、スプライトが現在フレームに固定される

#### Scenario: 石ころ転倒後のチャージ追跡開始時のアニメーション

- **WHEN** 石ころ転倒後に `startChasing()` が呼ばれ、敵がプレーヤーに到達する
- **THEN** 追跡中は走行アニメーションが継続して再生され、プレーヤーに到達した時点（`stopChasing()` 呼び出し時）に停止する
